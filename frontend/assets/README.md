# Assets Directory

This directory contains all static assets for the Weather API Application frontend.

## Directory Structure

```
assets/
├── images/          # Image files (icons, backgrounds, etc.)
├── fonts/           # Custom font files (if any)
├── icons/            # Icon files
└── data/             # Static data files (if any)
```

## Usage

- **Images**: Place any custom images, backgrounds, or graphics here
- **Icons**: Custom icon files (SVG, PNG) can be stored here
- **Fonts**: Custom font files if not using CDN fonts
- **Data**: Any static JSON or data files

## Notes

- Weather icons are currently loaded from OpenWeatherMap CDN
- Fonts are loaded from Google Fonts CDN
- Icons are loaded from Font Awesome CDN

If you want to use local assets instead of CDN resources, place them in the appropriate subdirectories and update the references in the HTML/CSS files.
